# Social VideoZz Downloader 🚀

A high-speed, GUI-based video downloader built with **Python**, **Tkinter**, and **yt-dlp**. It supports downloading videos from YouTube, Dailymotion, Facebook, Instagram, and many other platforms.

## ✨ Features
* **Multi-threaded Downloading:** Uses concurrent fragments to saturate your bandwidth for maximum speed.
* **Quality Selection:** Choose between 720p, 480p, 360p, and more.
* **Real-time Progress:** Visual progress bar with speed (MiB/s) and ETA tracking.
* **User Friendly:** Simple folder selection and URL input.
* **Broad Support:** Works with any site supported by `yt-dlp`.
